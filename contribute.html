Contribute to Trend Forge

We welcome contributions to Trend Forge, from code improvements to content suggestions. Whether you’re a developer, writer, or enthusiast, your input helps us deliver better insights.

How to Contribute

  1. Fork the Repository: Fork our GitHub repo.
  2. Clone Your Fork: 
    git clone https://github.com/your-username/trend-forge.git
  3. Create a Branch: 
    git checkout -b feature/your-feature
  4. Make Changes: Follow our Code of Conduct and use ESLint for code formatting.
  5. Commit: 
    git commit -m "Add your feature"
  6. Push: 
    git push origin feature/your-feature
  7. Open a Pull Request: Submit a PR with a clear description of your changes.

Contribution Types

- Code: Enhance the frontend (`/frontend`) or backend (`/backend`). - Content: Propose articles or translations via issues. - Bugs: Report issues on GitHub. - Feedback: Share ideas via Contact.

See our Documentation for technical details.
