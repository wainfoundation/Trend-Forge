Security

At Trend Forge, we prioritize the security of your data and transactions. Our platform employs industry-standard practices to protect your information and ensure safe interactions, including Pi Network payments.

Data Protection

- Encryption: All data transmissions use HTTPS with TLS 1.3, ensuring secure communication. - Storage: User data (e.g., contact form submissions) is stored in a secure PostgreSQL database with encrypted fields for sensitive information. - GDPR Compliance: We adhere to GDPR standards for data privacy, as outlined in our Privacy Policy.

Pi Network Payment Security

- Transaction Integrity: Subscriptions (e.g., 6 PI/month) are processed via the Pi Network API, with server-side validation to prevent tampering. - Wallet Protection: User Pi wallets are not stored on our servers; transactions occur directly through Pi Network’s secure infrastructure. - Fraud Prevention: We monitor for suspicious activity and comply with Pi Network’s KYC/KYB requirements.

User Account Security

- Authentication: Premium content access uses JWT-based authentication, with tokens securely stored in HTTP-only cookies. - Password Safety: Passwords (if implemented) are hashed using bcrypt. - Session Management: Sessions expire after inactivity, and users can log out to invalidate tokens.

For concerns or to report vulnerabilities, contact us at support@trendforge.com.
