Trend Forge Documentation

Welcome to the Trend Forge documentation. This guide covers how to use our platform, manage your subscription, make payments with Pi Network, and access developer resources for integrating with our services.

Getting Started

1. Sign Up or Log In

To access Trend Forge, you need a Pi Network account. Visit Sign Up to create an account or Log In with your existing Pi account. Authentication uses the Pi SDK for secure access.

2. Choose a Subscription

Select a subscription plan (Digital, Premium, All-Access, or Lifetime Access) on the Subscribe page. Payments are made in Pi cryptocurrency via your Pi wallet.

3. Explore Content

Browse sections like World, Business, Finance, Science, and Culture. Premium and All-Access subscribers enjoy exclusive content, ad-free browsing, and additional benefits.

Managing Your Subscription

Viewing Your Plan

Check your subscription status in your account settings (coming soon). You’ll see your plan (e.g., Premium, Lifetime Access) and renewal details.

Canceling a Subscription

Monthly plans (Digital, Premium, All-Access) can be canceled anytime via your account. Lifetime Access is a one-time payment with no recurring fees. Contact support for assistance.

Upgrading or Downgrading

To change plans, visit Subscribe and select a new plan. Your Pi wallet will be used to pay any price difference.

Making Payments with Pi Network

How Payments Work

Payments are processed via the Pi Network wallet in the Pi app. When you select a subscription plan, you’ll authenticate and confirm the payment amount (e.g., 6 PI for Digital) in the Pi app.

Troubleshooting Payments

Ensure you’re using the Pi Network app’s browser. If a payment fails, check your Pi wallet balance and network connectivity. Contact support if issues persist.

Pi Network Mainnet

Payments are subject to Pi Network’s mainnet availability. Currently, Trend Forge operates in sandbox mode for testing. Check Pi Network for mainnet updates.

Developer Resources

Pi SDK Integration

Trend Forge uses the Pi SDK for authentication and payments. Developers can integrate similar functionality using Pi SDK documentation. Example: 

Pi.init({ version: "2.0" });
Pi.authenticate(['payments', 'username']).then(auth => {
    console.log('Authenticated:', auth.user.username);
});

API Access

Trend Forge’s API (coming soon) allows developers to access content and user data. Register for API access via support.

Community Contributions

Contribute to Trend Forge’s open-source tools or suggest features via Contribute. Follow our Code of Conduct.
